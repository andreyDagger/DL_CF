Вычислите матричную функцию и её производную по заданному графу вычислений.

Входные данные

В первой строке содержится три целых положительных числа NN, MM, KK (2≤M+K≤N≤502≤M+K≤N≤50) — число вершин в графе вычислений, число входных параметров (вершин) и число выходных параметров (вершин). Далее следует NN строк — описание вершин графа вычислений. ii-я из этих строк содержит описание ii-й вершины:

  • var rr cc (1≤r,c≤251≤r,c≤25) — входной параметр функции, матрица состоящая из rr строк и cc столбцов.
  • tnh xx (1≤x<i1≤x<i) — матрица из значений гиперболического тангенса вычисленного от соответствующих компонент матрицы, полученной из xx-й вершины графа вычислений.
  • rlu α−1α−1 xx (1≤α−1≤1001≤α−1≤100, 1≤x<i1≤x<i) — матрица из значений функции параметрического линейного выпрямителя с параметром αα, вычисленной от соответствующих компонент матрицы полученной из xx-й вершины графа вычислений. α−1α−1 — целое число. Производная в нуле равна единице.
  • mul aa bb (1≤a,b<i1≤a,b<i) — произведение матриц, полученных из aa-й bb-й вершины графа вычислений соответственно.
  • sum lenlen u1u1 u2u2 ... ulenulen (1≤len≤101≤len≤10, ∀1≤j≤len:1≤uj<i∀1≤j≤len:1≤uj<i) — сумма матриц, полученных из вершин u1,u2,…,ulenu1,u2,…,ulen графа вычислений.
  • had lenlen u1u1 u2u2 ... ulenulen (1≤len≤101≤len≤10, ∀1≤j≤len:1≤uj<i∀1≤j≤len:1≤uj<i) — произведение Адамара (покомпонентное) матриц, полученных из вершин u1,u2,…,ulenu1,u2,…,ulen графа вычислений.

Гарантируется, что первые MM вершин и только они имеют тип var. Последние KK вершин считаются выходными. Никакие вершины не зависят от последних KK вершин. Гарантируется, что размеры матриц аргументов для каждой вершины согласованны.

Далее следует описание MM матриц — входных параметров соответствующих вершин графа вычислений в порядке возрастания их индексов.

Затем следует описание KK матриц — производных функции по соответствующим выходным вершинам в порядке возрастания их индексов.

Каждая строка каждой матрицы расположена на отдельной строке. Матрицы состоят из целых чисел по модулю не превышающих 10.

Выходные данные

Выведите KK матриц — значение параметров соответствующих выходных вершин графа вычисления в порядке возрастания их индексов. Затем выведите MM матриц — производных функции по соответствующим входным вершинам в порядке возрастания их индексов. Допустимая абсолютная и относительная погрешность 10−410−4.
